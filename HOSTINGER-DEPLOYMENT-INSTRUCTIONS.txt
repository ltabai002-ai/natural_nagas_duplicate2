================================================================================
  TROUBLESHOOTING
================================================================================

ISSUE: Website shows 404 error
SOLUTION:
   - Ensure index.html is in public_html root (not in subfolder)
   - Check file permissions: 644 for files, 755 for folders

ISSUE: Page refresh shows 404
SOLUTION:
   - Verify .htaccess file is uploaded and in public_html root
   - Check if mod_rewrite is enabled (contact Hostinger if needed)

ISSUE: Images not loading
SOLUTION:
   - Verify images folder is uploaded correctly
   - Check folder structure matches: images/[subfolder]/[image files]
   - Clear browser cache

ISSUE: Styles not working
SOLUTION:
   - Verify assets folder contains CSS and JS files
   - Check browser console for errors
   - Clear browser and server cache
